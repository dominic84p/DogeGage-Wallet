# Contributing to DogeGage Wallet

Thanks for your interest! Here's how you can help.

## Ways to Contribute

### Bug Reports
Found something broken? Open an issue with:
- What you expected to happen
- What actually happened
- Browser/device info
- Steps to reproduce

### Feature Suggestions
Have an idea? Open an issue and describe:
- What problem it solves
- How it should work
- Any examples from other wallets

### Code Contributions
1. Fork the repo
2. Create a branch (`git checkout -b fix/something`)
3. Make your changes
4. Test locally with `python3 -m http.server 8080`
5. Submit a PR

### Security Issues
**Do NOT open public issues for security vulnerabilities.**
See [SECURITY.md](SECURITY.md) for how to report them.

## Code Style

- Vanilla JS, no frameworks
- Keep it simple and readable
- Add comments for complex crypto operations
- Test on multiple browsers

## What We're Looking For

- Bug fixes
- Performance improvements
- Better error handling
- UI/UX improvements
- Documentation improvements
- New blockchain support (discuss first)

## What We're NOT Looking For

- Framework rewrites (React, Vue, etc.)
- Features that compromise privacy
- Custodial features
- Tracking/analytics
